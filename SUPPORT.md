# Support

CF Scanner is community-supported software.

- Use GitHub Discussions for usage questions, deployment help, and design proposals.
- Use GitHub Issues for reproducible bugs and accepted feature work.
- Use private security advisories for vulnerabilities.

Before requesting help, run `make check`, review logs, and consult [`docs/operations.md`](docs/operations.md). Remove credentials, public IP ownership details, and production data from all reports.

There is no guaranteed response time or commercial support commitment.
