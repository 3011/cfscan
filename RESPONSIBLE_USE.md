# Responsible Use

CF Scanner performs active network measurements. Use it only for systems and address ranges that you own or are explicitly authorized to test.

Operators are responsible for:

- complying with applicable law, contracts, provider policies, and acceptable-use rules;
- choosing safe concurrency, schedules, timeouts, and target ranges;
- avoiding disruption, evasion, unauthorized access, or abusive traffic;
- protecting Agent tokens, administrator sessions, database credentials, and collected data;
- providing a contact and stopping measurements when requested by an affected network owner.

The project does not grant permission to scan third-party infrastructure. Measurement results are provided without warranty and represent the network perspective of the selected Agent at the time of the test.
